# 🖼️ CNN Image Classification

A Convolutional Neural Network (CNN) for image classification developed using **PyTorch**. The project implements a complete deep learning pipeline, including data preprocessing, forward propagation, backpropagation, model training, and evaluation.

The model uses **ReLU** as the activation function for hidden layers, **Softmax** for output prediction, **Cross-Entropy Loss** as the loss function, and the **Adam Optimizer** for efficient parameter updates.

This project demonstrates the application of convolutional neural networks for image classification and provides a practical understanding of deep learning concepts using PyTorch.

---

## 🚀 Features

- Image Classification using CNN
- Built with PyTorch
- Convolutional Layers
- ReLU Activation
- Flatten Layer
- Fully Connected (Dense) Layer
- Softmax Output Layer
- Cross-Entropy Loss
- Adam Optimizer
- Forward Propagation
- Backpropagation
- Model Training and Evaluation
